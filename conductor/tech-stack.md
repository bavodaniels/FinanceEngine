# Tech Stack

## Core Technologies
- **Language:** Java 25 (OpenJDK 25.0.2)
- **Framework:** Spring Boot 4.0.2
- **Build Tool:** Gradle

## Data Persistence
- **Database:** PostgreSQL (Latest stable)
- **Migration Tool:** Liquibase or Flyway (TBD based on need)

## Testing & Quality
- **Testing Framework:** JUnit 5 (JUnit Platform)
- **Assertion Library:** AssertJ 3.27.7
- **Mocking:** Mockito (included in spring-boot-starter-test)

## Utilities
- **Math/Statistics:** Apache Commons Math 4.0-beta2 (for advanced features)
- **JSON Processing:** Jackson (default in Spring Boot)
